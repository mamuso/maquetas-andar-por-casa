flotaciones

Flotar es una de las cosas más básicas y más útiles a la hora de maquetar. El problema es que si no sabemos todos los truquitos acabamos flotándolo todo para que las cosas funcionen como nosotros queremos.

Cuando un elemento flota el contenido fluye a su alrededor para rodearlo. Al aplicar float el elemento responderá también a width y a height como si un elemento bloque.

Los valores de float pueden ser left, right o none.

Vamos a hacer un caso de uso típico para entender algunas de las soluciones más útiles para hacer flotaciones que se comporten tal y como queremos en todos los navegadores.
